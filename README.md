# Utaufile

#### 介绍
操作UTAU ust文件和袅袅虚拟歌手nn文件的python库

本python库依赖[numpy](https://numpy.org/) [mido](https://mido.readthedocs.io/en/latest/index.html)

#### 安装
> pip install utaufile

#### 功能

##### ust文件：
- 解析与写入ust文件
- 导出为mid、nn文件

##### nn文件：
- 解析与写入nn文件
- 导出为mid、ust文件
